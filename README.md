### 1. Клонировать репозиторий ###
### 2. Переименовать .env_example в .env ###
### 3. Создать виртуальное окружение командой python3 -m venv venv ###
### 4. Активировать виртуальное окружение командой . venv/bin/activate ###
### 5. Установить зависимости проекта командой pip install -r requirements.txt ###
### 6. В директории проекта выполнить docker compose up ###
### 7. Выполнить миграции БД командой python manage.py migrate ###
### 8. Запустить сервер разработки командой python manage.py runserver 0:8000 ###
### 9. Создать суперюзера для логина в админку командой python manage.py createsuperuser ###
